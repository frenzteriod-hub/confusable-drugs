Assessment Methodologies

Jaro-Winkler Distance Measures edit distances prioritizing characters near the start of the string (prefixes). Ideal for brand prefix matches. Configurable threshold (default: 0.82).
Levenshtein Edit Distance Computes the exact number of insertions, deletions, or substitutions required to transform name A into name B.
Soundex Phonetics Encodes strings into 4-character phonetic representations to capture words that sound similar (homophones).
Tall Man Lettering (TML) Applies ISMP/FDA-standard visual differentiation by capitalizing unique syllables in confusable pairs (e.g., traMADol vs. traZODone). Uses an official lookup map with algorithmic fallback for unknown pairs via longest common prefix/suffix alignment.
Drug Name Normalization Pre-processing pipeline that strips salt suffixes (hydrochloride, sulfate, sodium, etc.), removes special characters, and normalizes whitespace to isolate the active ingredient base name for accurate comparison.
ISMP Known LASA Overrides Hardcoded set of 13 clinically verified high-risk confusable pairs from the ISMP (Institute for Safe Medication Practices) reference list. These pairs always trigger alerts regardless of algorithmic scores.
Multi-Signal Composite Detection Combined scoring logic: flags a pair as LASA if Jaro-Winkler ≥ threshold, OR (Jaro-Winkler ≥ 0.75 AND Soundex ≥ 0.75), OR the pair exists in the ISMP clinical override list. Reduces false negatives by cross-validating orthographic and phonetic signals.

🇵🇭 Filipino-Aware Methodologies

Filipino Grapheme-to-Phoneme (G2P) Rule-based transformation engine modeling Tagalog and Bisaya pronunciation patterns. Applies 8 ordered phonological rules (digraphs first): ph→p, th→t, sh→s, f→p, v→b, z→s, plus Bisaya vowel collapse e→i and o→u. Transforms English drug names into their Filipino-realized pronunciation.
Filipino Phonetic Encoder Modified Soundex pipeline that first applies Filipino G2P transformation, then generates phonetic codes. Unlike standard Soundex (which preserves the first letter as-is), this captures how Filipino speakers actually pronounce the initial consonant — critical for detecting F/P and V/B merger confusions.
Filipino Multi-Signal Detection Cross-validates Filipino G2P-transformed Jaro-Winkler scores against Filipino Phonetic similarity. Classifies pairs as: Filipino-Only (confusable only in Filipino, missed by English systems), Amplified (more confusable in Filipino than English), or Standard (equally confusable in both).
