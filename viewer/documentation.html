Project goal

This project organizes Philippine FDA medicine records and proposes names that may look or sound alike. It helps a human reviewer find candidates faster. It does not diagnose, prescribe, or prove that a medication error will occur.

What data we collect

The product registry supplies registration number, published generic name, published brand name, strength, dosage form, classification, manufacturer, country, issuance date and expiry date. When an official leaflet is available, a separate process extracts sourced description and indication text with page references.

Important: 31,208 registration records do not mean 31,208 unique drugs. Strengths, forms, manufacturers, renewals and brands can create separate registrations.

Step-by-step workflow

  1. Preserve the source.Save each source separately and retain the registration number.
  2. Clean formatting.Standardize whitespace and capitalization without silently correcting published names.
  3. Create stable entities.Build generic-formulation and brand-name identifiers, linked back to registrations.
  4. Generate possible pairs.Use blocking keys so plausible names are compared without testing every possible combination.
  5. Build possible pronunciation paths.Keep the published spelling and create controlled Filipino-nativized, vowel-merged, casual-speech and stress variants.
  6. Calculate similarity signals.Measure spelling, edit distance, phonetic code, syllables, weighted phonemes, stress and letter bigrams.
  7. Separate obvious data cases.Place normalization duplicates and probable spelling/source variants in their own queues.
  8. Human review.Check ingredients, strengths, uses, registrations and external evidence before applying a final label.

Algorithms used

Jaro–Winkler

Scores spelling similarity from 0 to 1 and gives extra weight to a shared beginning.

Levenshtein edit distance

Counts how many character insertions, deletions or substitutions change one name into another.

Soundex

Creates a rough phonetic code. It is useful as a supporting signal, not proof of pronunciation.

Pronunciation lattice

Creates several controlled reading paths instead of declaring one pronunciation correct. Paths include the published spelling, Filipino nativization, possible e/i and o/u variation, casual-speech simplification and an alternative stress position.

Heuristic syllabification

Splits a comparison form around vowel nuclei, then aligns syllables. This helps inspect chunks of a spoken name rather than comparing letters alone.

Weighted phoneme alignment

Converts each path to a simple phoneme sequence. Controlled pairs such as f/p, v/b, z/s, e/i and o/u receive smaller substitution penalties than unrelated sounds.

Orthographic bigrams

Measures overlap between two-character chunks. It preserves a spelling signal alongside the experimental speech features.

Blocking

Groups names by prefixes, suffixes, length and phonetic keys before scoring. This makes a large dataset practical.

How the experimental score is combined

The dashboard exposes every component so a reviewer can see why a pair surfaced. The weights below are hand-set engineering choices for candidate ordering; they were not learned from labeled errors and must not be interpreted as accuracy or probability.

Pronunciation-path score

58% weighted phonemes + 32% syllable alignment + 10% stress position.

Multi-feature ensemble

42% path score + 23% phonemes + 15% syllables + 12% bigrams + 8% stress.

Numeric distinction penalty

When the letter-name base is almost identical but strength or version numbers differ, the final ensemble loses 0.10; it loses 0.18 when the letter base is exactly equal. Near-identical numeric variants stay in the full screening data but are moved out of the main high/medium review queue.

Why it is called experimental: no Filipino medicine-name confusion gold set has been used to estimate sensitivity, specificity, precision or recall. The interface therefore says “candidate,” never “verified,” based on this score.

Why are there two pair pages?

Pair Review

The complete high/medium human-review queue. Each pair can be retained through spelling, general phonetic, or experimental Filipino-aware evidence. Filipino-aware features already contribute to ranking here.

Filipino-aware

A focused explanation view containing only Pair Review candidates where at least one Filipino-aware signal contributed. It uses the same records and pair IDs; it is not a separate dataset or competing algorithm.

Review classifications

Verified LASA pair

Both medicines and a meaningful confusion risk have been confirmed by documented human review.

Potential LASA pair

Names are similar, but confusion evidence has not been established.

Spelling variant or probable typo

One published entry may be a misspelling or source-field problem requiring verification.

Normalization duplicate

The names become equal after harmless formatting differences are removed.

False positive

A human reviewer documented that the algorithmic match is unlikely to represent meaningful confusion.

Future experiments—not implemented or validated yet

Filipino medicine-pronunciation lexicon

Collect consented, speaker-provided pronunciations of medicine names instead of relying only on hand-written reading rules.

Learned ranking weights

Replace hand-set weights only after pharmacists create a sufficiently large, consistently labeled review set.

Speech-alternative evaluation

Measure whether multiple microphone transcripts improve retrieval without creating too many unrelated suggestions.

Ingredient and product guardrails

Use ingredient, strength, route and dosage form to explain product relationships and down-rank obvious same-product/version cases—not to declare LASA automatically.

Uncertainty calibration

After validation data exists, convert raw ranking scores into interpretable confidence bands with documented error rates.

Dialect and speaker variation study

Test whether controlled pronunciation paths represent real variation across Filipino speakers without assigning stereotypes to language groups.

Limitations and safety rules

  • A high similarity score does not prove clinical confusion.
  • The PH FDA source can contain misspellings, reversed fields or missing leaflets.
  • OCR text can be wrong and must retain a review flag and page reference.
  • An LLM may help organize text but must not invent missing indications or descriptions.
  • Pronunciation paths are hypotheses, not claims about how every Filipino speaker talks.
  • The fixed weights are not trained, calibrated or clinically validated.
  • Pairs are the review unit; groups only summarize multiple pair relationships.
