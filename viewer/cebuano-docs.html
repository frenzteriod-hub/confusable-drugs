Cebuano-aware · rule-based · experimental

Purpose and scope

The Cebuano-aware tool helps a reviewer search unfamiliar medicine names using approximate written readings and compare drug-name similarity. It keeps every official PH FDA name unchanged. Localized forms exist only as internal comparison paths and are never presented as corrected drug names.

What the score means: It is a screening and ranking value from 0 to 1. It is not a probability, diagnostic result, clinical risk estimate or proof that two medicines are a verified LASA pair.

The two tool modes

1. Find confusable drugs

The user may enter an official name or an approximate reading such as amoksisilin. The tool ranks up to five possible registered names and shows separate look-alike, sound-alike and overall scores. It does not silently select the first result. The user must choose the intended official name before its existing LASA candidates appear.

2. Compare two drugs

The user enters two official brand or generic names. If an entry is approximate, the tool offers ranked registered-name options and requires confirmation. It then displays spelling, localized sound, syllable and overall similarity signals.

How the algorithm works

  1. Load registered names.Use brand and generic names collected from the Philippine FDA Verification Portal. The compact browser index currently contains the project’s registered-name entities and precomputed review pairs.
  2. Normalize formatting.Convert text to lowercase and remove spaces and punctuation for comparison. The displayed official name remains unchanged.
  3. Build controlled reading paths.Keep the normalized spelling, then generate nativized, vowel-merged, casual-reading and selected initial-letter paths using the table below.
  4. Retrieve a shortlist.Use overlapping three-character chunks (trigrams) to find a manageable set of possible registered names instead of comparing the query with the entire name collection.
  5. Calculate four signals.Compare ordinary spelling, the strongest pair of reading paths, edit similarity and heuristic syllable chunks.
  6. Rank—not identify.Show up to five possible registered names with their scores. The user chooses the intended official medicine; the program does not claim that rank 1 is correct.
  7. Show LASA candidates.After selection, retrieve the chosen name’s precomputed high- or medium-priority review candidates and display visual, sound and rank scores.

Similarity algorithms and scores

Looks alike

Jaro–Winkler similarity compares the normalized query with the published name. It rewards matching characters and a shared beginning while tolerating selected transpositions.

Sounds alike

Generate all controlled paths for the query and registered name, compare every path pair with Jaro–Winkler, and retain the strongest path-pair score. The selected internal paths are not shown to the user.

Edit similarity

Use normalized Levenshtein similarity on the strongest reading-path pair. This tolerates missing, inserted or substituted characters that can represent part of a misread syllable.

Syllable similarity

Split the selected paths heuristically around vowel nuclei, then align the resulting chunks. This is an engineering approximation, not a validated Cebuano syllabifier.

Input-interpretation formula

overall = 0.30 × looks + 0.35 × sounds + 0.20 × edit + 0.15 × syllables

The weights are hand-set for this prototype. They were not learned from labeled Cebuano pronunciation data.

Confusable-candidate rank

rank = max(spelling, edit, Filipino-transformed spelling, experimental ensemble)

The review queue keeps only algorithmic high- and medium-priority candidates and limits how many pairs each indexed name contributes. Its displayed sound score is the precomputed pronunciation-lattice score. None of these values is a clinical probability.

Why these letters and pronunciations?

Direct Cebuano evidence

Wolff’s phonology describes a three-vowel analysis using i, a, u, with inconsistent written e/i and o/u; it also describes j/dy, c/ts, vowel length and several glide patterns. Those observations motivate controlled alternative paths rather than replacing official spellings.

Inventory-motivated paths

The inventory contains sounds such as /p/, /b/, /t/ and /s/ but does not list several sounds common in English drug names. The prototype tests possible substitutions, but an absent inventory item does not prove that every Cebuano speaker substitutes it.

Engineering heuristics

Rules for English letter groups, soft c/g, silent or reduced h, and initial i/ay are search-tolerance hypotheses. They improve retrieval for selected examples but require speaker studies before being described as Cebuano pronunciation rules.

Why keep multiple paths?

Medicine names are unfamiliar technical forms and may be read through English, Filipino, Cebuano or spelling-based strategies. Retaining several controlled paths avoids declaring one localized form correct and lets the strongest plausible comparison contribute to ranking.

Limitations and required validation

  • The rule set does not represent every Cebuano dialect, speaker, age group or level of English exposure.
  • The Wolff dictionary was published in 1972 and is not a medicine-pronunciation dataset.
  • Several transformations are engineering heuristics, not linguistic findings.
  • Jaro–Winkler and Levenshtein measure string similarity; they do not model human medication errors.
  • The syllabifier and weights are hand-built and have not been calibrated against labeled Cebuano recordings.
  • High-scoring options may be false positives. Users must confirm the official name, ingredient, strength and dosage form.
  • “Verified LASA” and “False positive” labels require documented human and pharmacist review.
